# Python学生信息管理系统 python690

本项目是基于Python的学生信息管理系统，包括源码、运行文档、PPT以及开发文档。系统具备完善的功能模块，致力于提供便捷的信息管理服务。

## 技术栈

- **开发工具**：Vscode、Navicat
- **运行环境**：Python 3.0以上、MySQL 8.0
- **前端技术**：HTML、CSS、JavaScript、jQuery
- **后端技术**：Python、Flask

## 功能模块

- 登录
- 注册
- 学生管理
- 成绩管理
- 班级管理
- 课程管理
- 宿舍管理
- 任务管理
- 资料管理
- 意见反馈
- 系统管理

## 系统角色

- 管理员：负责系统的管理与维护
- 教师：负责成绩录入、课程管理等
- 学生：查询成绩、个人信息管理等

## 目录结构

```
project/
│
├── app/               # 应用目录
│   ├── static/        # 静态文件目录
│   └── templates/     # 模板文件目录
│
├── docs/              # 文档目录
│
├── src/               # 源代码目录
│   ├── models/        # 数据模型目录
│   ├── views/         # 视图处理目录
│   └── utils/         # 工具函数目录
│
└── venv/              # 虚拟环境目录
```

## 核心亮点

- **灵活的功能模块设计**：各管理模块相对独立，便于扩展和维护。
- **稳定的技术框架**：基于成熟的Flask框架，确保系统运行稳定。
- **简洁的界面设计**：使用HTML、CSS进行界面设计，简洁直观，易于操作。

## 部署步骤

1. 确保Python 3.0以上和MySQL 8.0环境安装正确。
2. 使用Navicat导入数据库文件。
3. 解压项目到指定的目录。
4. 在项目根目录下运行虚拟环境。
5. 安装依赖包。
6. 运行应用。
